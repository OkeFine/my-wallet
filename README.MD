# Ronin Wallet - FE Challenge

## Front End

- Build by Create React App
- How to run:

```
cd /app
cp .env.sample .env
yarn install
yarn start
```

## Backend End

- Build by Nodejs / Expressjs
- How to run:

```
cd /server
cp .env.sample .env
yarn install
yarn start
```
